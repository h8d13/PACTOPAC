# pactopac - hide store 

```
git clone https://github.com/h8d13/PACTOPAC
cd PACTOPAC
sudo pacman -S python-gobject pacman-contrib vte4 gtk4 libadwaita
sudo python3 main.py
```
---
Can add an alias like `alias pacm='sudo -b python path/to/main.py; exit'`
