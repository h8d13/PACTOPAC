# Making Arch accessible to anyone <3

## PacToPac 

Simple GUI for pacman/flatpak using Subprocess. 

---

### Get it running:

`$ pacman -S python-gobject gtk4 libadwaita`

`$ sudo python3 main.py`

### What ? 

**Settings:**

Enable multi-lib
Mirrors
Detect Hardware
Flatpak 
Dark/Light mode

**Core features:**
Subprocess display
Package info/install/remove/clean
Search
